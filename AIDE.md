## Aléatoire

Nombre aléatoire entre 0 et 1 : 

```javascript
Math.random()
```

Nombre aléatoire entier entre 0 et MAX : 

```javascript
function getRandomInt(max) {
  return Math.floor(Math.random() * Math.floor(max));
}
```
